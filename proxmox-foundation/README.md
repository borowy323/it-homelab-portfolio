### Issue: VM had no internet access
- Cause: incorrect bridge configuration

### Issue: DNS not resolving
- Cause: wrong DNS server
- Fix: updated DNS configuration

### Issue: SSH connection failed
- Cause: service not running
- Fix: enabled SSH service
